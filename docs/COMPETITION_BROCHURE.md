## 二、核心功能（7大模块）

### 2.1 📄 数据集管理 — 多种导入方式

- 文件上传：Excel/CSV/TXT/JSON/Markdown/日志，拖拽即传
- 数据库连接：MySQL/PostgreSQL/Oracle/SQL Server，一键导入
- 剪贴板粘贴：快速测试小量数据
- 自动识别编码和格式

### 2.2 🔍 敏感数据识别 — 多语言智能引擎

- 6种语言自动检测：中/英/日/韩/法/德
- 正则+关键词+格式校验三层匹配
- 内置丰富识别规则，支持自定义
- 识别结果可视化（饼图+柱状图+明细）
- 一键跳转脱敏（识别→脱敏闭环）

### 2.3 🔒 数据脱敏 — 独创关联仿真算法

**四种脱敏策略：**
| 策略 | 效果 | 适用场景 |
|------|------|---------|
| 仿真 | 生成格式合理的假数据 | 姓名、手机号 |
| 关联仿真 ⭐ | 相同原值→相同假值（跨表一致） | 多表关联场景 |
| 部分遮盖 | 保留首尾、中间打星 | 手机号、身份证 |
| 完全遮盖 | 全部替换 | 高敏感字段 |

**独创关联仿真：**
- 基于 HMAC-SHA256 的确定性脱敏算法
- 30组独立密钥实现业务隔离
- 输入相同、密钥相同 → 输出始终相同
- 保证跨表关联数据的一致性

## 三、技术创新点

### 3.1 🥇 多语言智能识别引擎

传统识别工具通常只支持单一语言。本平台独创**Unicode字符集特征+langdetect**双重检测机制，首次将中文/英文/日文/韩文/法文/德文6种语言的敏感信息识别集于一体，特别支持中日韩多语言混排场景。

### 3.2 🥇 关联仿真脱敏（确定性脱敏）

传统随机脱敏会导致跨表关联断裂（如张三在表A变成李四、在表B变成王五）。本平台独创的**基于HMAC-SHA256的确定性仿真算法**，保证相同原始数据使用相同密钥时，始终输出相同的假数据，从而**完整保护数据关联关系**。30组独立密钥满足不同业务隔离需求。

### 3.3 🥇 全流程闭环与智能跳转

识别结果 → 一键跳转脱敏任务 → 自动传递数据集和推荐规则。不同于传统工具"识别"和"脱敏"分离的割裂体验，本平台实现**从数据导入到脱敏报告的无缝衔接**。

### 3.5 🥇 AI大模型智能识别与脱敏

本平台集成 DeepSeek、OpenAI、千问、Kimi、GLM 等多种大语言模型，实现**语义级的敏感信息识别**。与传统规则引擎不同，AI引擎能理解上下文语义，识别"王总上月转账50万"这类规则无法捕捉的隐式敏感信息。

**核心特性：**
- 多模型统一API调用架构，按需切换
- AI识别 + 人工复核双保险闭环
- 推理过程可追溯，附带法规依据
- 支持联网搜索增强识别准确性

### 3.6 🥇 四套完整UI主题架构

不是简单的颜色切换，而是**四套独立的组件体系**，通过 `ThemeLoader` 动态加载，实现整体视觉方案的完整替换，包括：
- 侧边栏布局和配色
- 卡片/表格/按钮/输入框风格
- 全局配色和字体方案
- 每一处细节都经过独立设计

### 3.7 🥇 零代码操作体验

从数据上传到报告导出，所有操作均通过图形化界面完成。用户无需了解正则表达式、SQL语句或任何编程知识，鼠标点击即可完成从识别到脱敏的全部流程。
