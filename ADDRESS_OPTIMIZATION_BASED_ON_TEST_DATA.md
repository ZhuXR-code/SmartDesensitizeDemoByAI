# 基于真实测试数据的地址识别优化

## 📊 测试数据分析

### 1. 地址格式多样性

| 语言 | 示例 | 特征 |
|------|------|------|
| 中文 | 广西壮族自治区...x座 917806 | 省市区+路+座+邮编 |
| 英文 | 72658 Braun Springs, Forbesview, NY 94223 | 数字+地名+州缩写+邮编 |
| 日文 | 長崎県横浜市港南区白金台31丁目18番17号 | 県市区+地名+丁目+番+号 |
| 韩文 | 대전광역시 송파구 언주65가 937-79 | 광역시+구+地名+가+邮编 |
| 法文 | 241, boulevard Victoire Da Silva... | 数字+boulevard+人名 |
| 德文 | Bettina-Hermann-Straße 257... | 人名-Straße+数字 |

### 2. 关键发现

#### a) 中文地址特征
- 包含"座"、"栋"、"号"等建筑标识
- 如：`x座`、`d座`、`h座`、`M座`、`U座`
- 邮编通常是6位数字

#### b) 韩文地址特征
- 必须包含：`광역시`（广域市）、`구`（区）、`동`（洞）
- 如：`대전광역시 송파구`
- 邮编格式：`937-79`

#### c) 日文地址特征
- 必须包含：`県`、`市`、`区`、`丁目`、`番`
- 如：`長崎県横浜市港南区白金台31丁目18番17号`
- 地址结构：県→市→区→地名→丁目→番→号

#### d) 德文地址特征
- 使用`-Straße`模式：`Bettina-Hermann-Straße`
- 或`Str.`缩写
- 如：`Bettina-Hermann-Straße 257, 95961 Eichstätt`

#### e) 英文地址特征
- 州缩写：`NY`、`CA`、`TX`等
- 邮编：5位或5+4位（如`94223`或`94223-1234`）
- 如：`Forbesview, NY 94223`

### 3. 姓名列分析

| 语言 | 示例 | 特征 |
|------|------|------|
| 中文 | 薛慧、钱桂芝 | 2-4个汉字 |
| 英文 | Wanda Bell、Martin Blot | 2个英文单词，首字母大写 |
| 日文 | 小川 香香、吉田 拓真 | 2-3个日文词，含空格 |
| 韩文 | 이 경 자、안 태 진 | 3个韩文词，含空格 |
| 法文 | Vincent Lamy、Caroline Pritchard | 2个法文单词 |
| 德文 | Anto Jungfer、Bettina-Hermann-Straße | 含连字符 |

## 🔧 优化策略

### 1. 地址识别增强

#### 中文地址
```python
# 新增关键词
'座', '栋', '幢', '号楼', '门牌'
```

#### 韩文地址
```python
# 必须包含以下之一：
'광역시', '특별시', '도', '시', '구', '동', '리'
# 邮编模式
r'\d{3}-\d{2}'  # 如 937-79
```

#### 日文地址
```python
# 必须包含以下之一：
'県', '都', '道', '府'
# 地址结构
r'\d+丁目'  # 如 31丁目
r'\d+番'    # 如 18番
r'\d+号'    # 如 17号
```

#### 德文地址
```python
# 必须包含以下之一：
'-Straße', '-strasse', 'Str.'
# 邮编模式（德国邮编5位）
r'\b\d{5}\b'
```

#### 英文地址
```python
# 州缩写（50个州）
'NY', 'CA', 'TX', 'FL', 'IL', ...
# 邮编模式
r'\b\d{5}(-\d{4})?\b'
```

### 2. 姓名识别增强

#### 避免误识别

**英文姓名 vs 英文地址：**
- 姓名：`Wanda Bell` - 没有数字、没有地址关键词
- 地址：`72658 Braun Springs, NY 94223` - 有数字、有州缩写、有邮编

**判断逻辑：**
```python
def is_name(text):
    # 如果包含数字，可能是地址
    if re.search(r'\d+', text):
        return False
    
    # 如果包含地址关键词，不是姓名
    if has_address_keywords(text):
        return False
    
    # 英文名：2-3个单词，首字母大写
    words = text.split()
    if len(words) == 2 and all(w[0].isupper() for w in words):
        return True
    
    return False
```

**韩文姓名 vs 韩文地址：**
- 姓名：`이 경 자` - 3个词，有空格，无地址关键词
- 地址：`대전광역시 송파구` - 有`광역시`、`구`

**判断逻辑：**
```python
def is_korean_name(text):
    # 如果包含地址关键词，不是姓名
    if any(kw in text for kw in KOREAN_ADDRESS_KEYWORDS):
        return False
    
    # 韩文姓名：通常3个词，有空格
    words = text.split()
    if len(words) == 3 and all(is_korean(w) for w in words):
        return True
    
    return False
```
