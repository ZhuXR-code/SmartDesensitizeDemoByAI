# 敏感信息智能脱敏平台 - 安装说明

## 快速选择

| 你的情况 | 推荐方案 | 难度 | 前置要求 |
|---------|---------|------|---------|
| 已安装 Docker | [Docker 一键部署](#方案一-docker-一键部署) | ⭐ | Docker Desktop |
| 有 Python，无 Docker | [便携式部署](#方案二-便携式部署) | ⭐⭐ | Python 3.10+ |
| 纯 Windows 小白用户 | [Windows 安装包](#方案三-windows-安装包) | ⭐ | 无 |

## 方案一：Docker 一键部署

> 适合大部分用户，只需安装 Docker Desktop 即可。

### 第1步：安装 Docker Desktop

下载并安装 Docker Desktop：https://www.docker.com/products/docker-desktop/

安装后重启电脑，确保 Docker 图标在任务栏显示。

### 第2步：一键启动

进入 `docker-deploy` 目录，双击 `一键部署.bat`（Windows）或运行：

```bash
cd docker-deploy
# Linux/Mac
chmod +x deploy.sh && ./deploy.sh
```

### 第3步：访问平台

打开浏览器访问：**http://localhost:8080**

---

## 方案二：便携式部署

> 适合已有 Python 环境的用户，无需安装 Docker 和 MySQL。

### 第1步：安装 Python

下载安装 Python 3.10+：https://www.python.org/downloads/

安装时**务必勾选** "Add Python to PATH"。

### 第2步：启动平台

进入 `portable` 目录，双击 `启动平台.bat`（Windows）或运行：

```bash
cd portable
chmod +x start.sh && ./start.sh
```

### 第3步：访问平台

打开浏览器访问：**http://localhost:5173**

---

## 方案三：Windows 安装包

> 适合完全不熟悉技术的用户，无需安装任何环境。

1. 从发布页面下载 `敏感信息脱敏平台_Setup.exe`
2. 双击运行，按提示完成安装
3. 双击桌面快捷方式即可启动

## 常见问题

### Q: 启动后页面打不开？

1. 检查 Docker 是否正常运行（Docker 方案）
2. 检查端口是否被占用：`netstat -ano | findstr :8080`
3. 查看服务日志排查错误

### Q: 如何停止服务？

- Docker 方案：`docker compose down`
- 便携方案：按 `Ctrl+C`

### Q: 数据会丢失吗？

- Docker 方案：数据存储在 Docker 卷中，执行 `docker compose down -v` 会删除数据
- 便携方案：数据存储在 `portable/backend/data/` 目录

### Q: 如何更新版本？

下载最新安装包，覆盖旧文件即可。数据文件不会被覆盖。
