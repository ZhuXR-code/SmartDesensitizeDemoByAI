# 部署说明

## 前置准备

### 克隆项目

```bash
git clone <项目地址>
cd code03
```

### 目录说明

```
code03/
├── backend/           # Python FastAPI 后端
├── frontend/          # Vue3 前端
├── docs/              # 项目文档
├── deploy/            # Docker 部署配置文件（供开发/运维参考）
└── distribution/      # 安装包文件（供最终用户使用）
    ├── docker-deploy/     # Docker 一键部署
    ├── portable/          # 便携式部署（SQLite）
    ├── windows-installer/ # Windows 安装包
    ├── README.md          # 安装包总览
    └── 安装说明.md        # 快速入门指南
```

## 部署方案

详见各子目录内的说明文档：
- [安装说明.md](./安装说明.md) - 快速入门
- [docker-deploy/README.md](./docker-deploy/README.md) - Docker 部署
- [portable/README.md](./portable/README.md) - 便携式部署
- [windows-installer/README.md](./windows-installer/README.md) - Windows 安装包

## 注意事项

1. **Docker 方案**需先安装 Docker Desktop
2. **便携方案**需先安装 Python 3.10+
3. 上传文件默认存储在 `uploads/` 目录，请注意备份
4. 数据库默认存储在 Docker 卷或本地文件，请注意定期备份
