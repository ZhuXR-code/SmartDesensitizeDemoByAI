# 创建脱敏任务 - 分层选择功能说明

## 📋 功能概述

在创建脱敏任务时，脱敏规则的选择改为**两步分层选择**：

1. **第一步**：选择脱敏方式（完全遮盖 / 仿真造数 / 部分遮盖）
2. **第二步**：根据选择的脱敏方式，筛选并选择具体的脱敏规则

## 🎯 使用流程

### 步骤演示

```
┌─────────────────────────────────────────────────────┐
│ 字段名     │ 语言 │ 脱敏方式      │ 脱敏规则        │
├────────────┼──────┼───────────────┼────────────────┤
│ 姓名       │ 中文 │ [选择方式 ▼]  │ (禁用)         │
│            │      │               │                │
│ 选择后:    │      │ 🔴 完全遮盖   │ [选择规则 ▼]   │
│            │      │               │ • 完全遮盖     │
│            │      │               │   张三 → *     │
└────────────┴──────┴───────────────┴────────────────┘
```

### 详细步骤

1. **进入创建脱敏任务页面**
   - 选择数据集
   - 点击某一行，自动跳转到"配置规则"步骤

2. **配置字段规则**
   
   对于每个字段，需要依次选择：
   
   **① 选择脱敏方式**
   ```
   ┌──────────────────┐
   │ 选择方式       ▼ │
   ├──────────────────┤
   │ 🔴 完全遮盖      │  ← 点击选择
   │ 🟢 仿真造数      │
   │ 🟡 部分遮盖      │
   └──────────────────┘
   ```
   
   **② 选择具体规则**
   ```
   选择"完全遮盖"后：
   ┌──────────────────┐
   │ 选择规则       ▼ │
   ├──────────────────┤
   │ 完全遮盖         │  ← 只有一个选项
   │   张三 → *       │
   └──────────────────┘
   
   选择"仿真造数"后（以中文为例）：
   ┌──────────────────────┐
   │ 选择规则           ▼ │
   ├──────────────────────┤
   │ 姓名仿真             │
   │   张三 → 李伟        │
   │ 手机号仿真           │
   │   138... → 139...   │
   │ 身份证号仿真         │
   │   110... → 110...   │
   │ ...                  │
   └──────────────────────┘
   ```

3. **预览确认**
   - 查看脱敏效果预览
   - 确认无误后点击"开始脱敏"

## 💡 功能特点

### 1. 智能筛选

- 选择脱敏方式后，规则下拉框**只显示该方式下的规则**
- 同时还会**根据字段语言进一步筛选**
  - 例如：中文字段 + 仿真造数 → 只显示中文仿真规则

### 2. 实时预览

- 每个规则选项都显示**脱敏前后对比**
  ```
  姓名仿真
  张三 → 李伟
  
  手机号部分遮盖
  13800138000 → 138****8000
  ```

### 3. 联动清空

- 切换脱敏方式时，**自动清空**已选择的具体规则
- 避免方式与规则不匹配的情况

### 4. 禁用保护

- 未选择脱敏方式时，规则下拉框**处于禁用状态**
- 提示文字："先选脱敏方式"

## ⚙️ 技术实现

### 前端代码结构

```javascript
// 字段配置数据结构
{
  column: 'name',                    // 字段名
  language: 'zh',                    // 检测语言
  desensitization_method: 'simulation', // 脱敏方式（新增）
  rule_id: 2                         // 具体规则ID
}

// 筛选函数
const getRulesByMethodAndLanguage = (method, lang) => {
  return rules.value.filter(r => {
    const methodMatch = r.desensitization_method === method
    const langMatch = r.language === lang || r.language === 'all'
    return methodMatch && langMatch
  })
}

// 切换脱敏方式时的处理
const onMethodChange = (row) => {
  row.rule_id = null  // 清空已选规则
}
```
