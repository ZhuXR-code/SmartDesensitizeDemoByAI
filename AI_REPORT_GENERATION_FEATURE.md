# AI检测报告生成功能说明

## 功能概述

本次更新为AI智能检测添加了完整的报告生成功能，支持HTML和Markdown两种格式。

## 一、功能特性

### 1. 风险列宽度优化
- **调整前**: 60px（标签显示不完整）
- **调整后**: 80px（完整显示high/moderate/low）

### 2. 置信度悬浮提示
鼠标悬停在置信度标签上时，显示详细说明：
```
置信度说明：
AI模型对检测结果的可信程度
• 90%-100%：高度确信
• 70%-89%：比较确信
• 50%-69%：一般确信
• <50%：较低确信，建议人工复核
```

### 3. 报告生成功能

#### 支持的格式
- ✅ **Excel** - 原有的表格导出功能
- ✅ **HTML** - 精美的可视化报告（新增）
- ✅ **Markdown** - 简洁的文本报告（新增）

#### HTML报告特点
- 🎨 渐变色彩设计，专业美观
- 📊 统计卡片展示关键指标
- 📈 敏感类型分布图表
- 📋 详细的检测结果表格
- 📱 响应式设计，适配各种屏幕
- 🔍 悬停显示完整内容

#### Markdown报告特点
- 📝 简洁清晰的文本格式
- 📊 表格形式展示统计数据
- 💻 易于版本控制和分享
- 🔧 可轻松转换为其他格式

## 二、使用方法

### 前端操作

1. 进入"AI智能识别"页面
2. 在"检测任务列表"中点击任意任务的"详情"按钮
3. 在详情对话框底部，可以看到三个按钮：
   - **导出Excel** - 生成.xlsx格式的表格文件
   - **生成HTML报告** - 生成精美的HTML可视化报告
   - **生成Markdown报告** - 生成.md格式的文本报告
4. 点击任一按钮，系统会生成报告并自动下载

### 后端API

#### 1. 生成HTML报告
```
GET /api/ai/detect/tasks/{task_id}/report/html
```

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "file_path": "uploads/reports/ai_detection_report_123_1234567890.html",
    "message": "HTML报告生成成功"
  }
}
```

#### 2. 生成Markdown报告
```
GET /api/ai/detect/tasks/{task_id}/report/markdown
```

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "file_path": "uploads/reports/ai_detection_report_123_1234567890.md",
    "message": "Markdown报告生成成功"
  }
}
```

## 三、报告内容

### 报告包含的信息

1. **基本信息**
   - 任务名称
   - 数据集名称
   - 生成时间
   - 任务ID

2. **检测概览**
   - 总行数
   - 发现敏感数据数量
   - 高风险/中风险/低风险数量
   - 处理耗时

3. **敏感类型分布**
   - 按敏感类型统计数量
   - 排序显示（从多到少）

4. **检测结果详情**
   - 行号
   - 列名
   - 原始值（HTML中限制长度，悬停查看完整）
   - 敏感类型
   - 置信度
   - 风险等级
   - 法规依据

## 四、技术实现

### 后端实现

**文件**: `backend/app/services/report_generator.py`

核心类 `ReportGenerator` 提供两个方法：

1. `generate_html_report(task, results)` - 生成HTML报告
   - 使用内联CSS样式，无需外部依赖
   - 渐变色彩设计
   - 响应式布局
   - 限制显示前100条结果（避免文件过大）

2. `generate_markdown_report(task, results)` - 生成Markdown报告
   - 标准Markdown语法
   - 表格形式展示
   - 限制显示前100条结果

### 前端实现

**文件**: `frontend/src/views/ai/AiDetection.vue`

新增函数 `generateReport(taskId, format)`:
- 根据format参数调用不同的API
- 生成成功后自动下载文件
- 显示友好的提示信息

## 五、文件存储

所有报告文件保存在：
```
uploads/reports/
├── ai_detection_123_1234567890.xlsx      # Excel导出
├── ai_detection_report_123_1234567890.html  # HTML报告
└── ai_detection_report_123_1234567890.md    # Markdown报告
```

文件名格式：
- Excel: `ai_detection_{task_id}_{timestamp}.xlsx`
- HTML: `ai_detection_report_{task_id}_{timestamp}.html`
- Markdown: `ai_detection_report_{task_id}_{timestamp}.md`

## 六、注意事项

1. **结果数量限制**
   - HTML和Markdown报告只显示前100条结果
   - 如需查看全部结果，请使用Excel导出

2. **文件大小**
   - HTML报告由于包含样式，文件较大（通常100-500KB）
   - Markdown报告文件较小（通常10-50KB）
   - Excel文件大小适中

3. **浏览器兼容**
   - HTML报告在现代浏览器中显示最佳
   - 建议使用Chrome、Firefox、Edge等浏览器查看

4. **性能考虑**
   - 大数据集生成报告可能需要几秒时间
   - 建议在任务完成后再生成报告
